About Me

My name is Diana. My tech journey started in high school when I attended a summer class to learn how to code games using Scratch. Back then, I had fun making a game called 'Falling Bananas', yet it did not cross my mind to consider tech as my career.
By the time I was in college majoring in Public Relations, I had the opportunity to take a Web Design class and learned HTML, CSS and a little bit of Javascript. Since I only had a year left and not enough time to take more computer science classes, I decided took the opportunity to become the Teacher Assistant for the following semester.
When I moved back to the city, I looked for opportunities to learn how to code and came across Pursuit. I attended Pursuit's coding program and learned Full-Stack Web Development. At Pursuit, I learned Javascript, Node.js, Express, PostgreSQL and React to build full-stack web applications.
I love how coding challenges me to think outside of the box and provides me with complex problems to solve as well as give me the freedom to fledge out my ideas creativily by building out user interfaces for the Web. This is the beginning of my career, and I look forward to learning more about coding and the tech industry.
